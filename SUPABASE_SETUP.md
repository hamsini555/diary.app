# Supabase Setup Guide

## Step 1: Create a Supabase Project
1. Go to [supabase.com](https://supabase.com)
2. Sign up or log in
3. Click "New Project"
4. Enter project details and create the project
5. Wait for the project to initialize

## Step 2: Get Your Credentials
1. In your Supabase dashboard, go to **Settings** → **API**
2. Copy your **Project URL** (this is your SUPABASE_URL)
3. Copy your **anon public** key (this is your SUPABASE_ANON_KEY)

## Step 3: Create the Database Table
1. In Supabase, go to the **SQL Editor** tab
2. Click **New Query** and run this SQL:

```sql
CREATE TABLE entries (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  title TEXT NOT NULL,
  content TEXT NOT NULL,
  mood TEXT NOT NULL,
  datetime TEXT NOT NULL,
  password TEXT NOT NULL,
  created_at TIMESTAMP DEFAULT NOW()
);
```

3. Click **Run** to create the table

## Step 4: Configure Your App
1. In your project root, create a `.env` file (copy from `.env.example`)
2. Paste your SUPABASE_URL and SUPABASE_ANON_KEY
3. Save the file

## Step 5: Install Dependencies and Run
```bash
npm install
npm start
```

Your app will now use Supabase instead of SQLite!

## Notes
- Keep your `.env` file private (it's already in .gitignore if using git)
- The `password` field in the database stores plain text (consider hashing in production)
- Supabase provides 500MB of free database space on the free tier
